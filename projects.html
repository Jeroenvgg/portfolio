Asangue

Asangue is a text-based maffia game which is currently in the development phase. In this game you will be able to steal cars, do crimes, own casinos and interact with other players. The goal is to have fun and rise to the top. I'm blending this project in with my master thesis project where I will link NFT technology to this game. In the game you will be able to mint your own guard-dog;directly on the blockchain. The guard-dog will have in-game advantages and you can transfer this in the game aswell as import or export it out the game. If the website is build it will be displayed HERE.
